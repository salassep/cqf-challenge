# CQF Challenge
CQF challenge repository.

## Tech Stack
- Vue
- Vite
- Tailwind

## Run Locally
Clone the project.
```bash
  git clone https://github.com/salassep/cqf-challenge.git project_name
```

Go to project directory:
```bash
  cd project_name
```

Install dependencies:
```bash
  npm install
```

Run in development mode:
```bash
  npm run dev
```
